# Overview

> Documentation coming soon.

**orka'i** is a self-hosted PaaS that runs on [K3s](https://k3s.io) — real Kubernetes under the hood, not Docker wrappers. Deploy apps, managed databases, and cron jobs from a single control plane.

## What you can do

- **Deploy apps** — git-push or Docker image deploys with in-cluster Kaniko builds
- **Run databases** — managed PostgreSQL, MySQL, Redis, and more
- **Schedule jobs** — native Kubernetes CronJobs
- **Inspect the cluster** — nodes, Helm releases, Traefik ingress, live logs
