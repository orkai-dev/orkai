# Visão geral

> Documentação em breve.

**orka'i** é um PaaS self-hosted que roda em [K3s](https://k3s.io) — Kubernetes de verdade por baixo, não wrappers de Docker. Faça deploy de apps, bancos de dados gerenciados e cron jobs a partir de um único plano de controle.

## O que você pode fazer

- **Fazer deploy de apps** — git-push ou deploy por imagem Docker com builds Kaniko no cluster
- **Rodar bancos de dados** — PostgreSQL, MySQL, Redis gerenciados e mais
- **Agendar jobs** — CronJobs nativos do Kubernetes
- **Inspecionar o cluster** — nós, releases Helm, ingress Traefik, logs ao vivo
