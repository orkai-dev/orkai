# Resumen

> Documentación próximamente.

**orka'i** es un PaaS autoalojado que corre en [K3s](https://k3s.io) — Kubernetes real bajo el capó, no wrappers de Docker. Despliega apps, bases de datos gestionadas y cron jobs desde un único plano de control.

## Qué puedes hacer

- **Desplegar apps** — git-push o despliegues por imagen Docker con builds Kaniko en el clúster
- **Ejecutar bases de datos** — PostgreSQL, MySQL, Redis gestionados y más
- **Programar jobs** — CronJobs nativos de Kubernetes
- **Inspeccionar el clúster** — nodos, releases Helm, ingress Traefik, logs en vivo
